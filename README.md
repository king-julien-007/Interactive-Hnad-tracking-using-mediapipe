Mediapipe Interactive handtracking solution

A real-time hand tracking visual effects project built with **React**, **TypeScript**, **MediaPipe**, and **Canvas/WebGL-style post-processing**.

## Features

- Real-time hand tracking using webcam
- Multiple visual effect modes
- Dual hand interaction regions
- Adjustable effect controls
- Live camera-based rendering
- Built with a modern React + TypeScript setup

## Tech Stack

- React
- TypeScript
- MediaPipe Hand Landmarker
- HTML5 Canvas
- CSS
